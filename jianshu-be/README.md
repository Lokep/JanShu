
## Description

remake the JianShu Application via [Nest](https://github.com/nestjs/nest) ,it's a typescript framework,
this repository is only for the back-end apis.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
